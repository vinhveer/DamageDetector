# damage_scan outputs

This folder is the default location for the `damage_scan` SQLite database when running:

`python -m object_detection.damage_scan`

or the wrapper script:

`python semi-labeling/step1_gdino_labeling/run_damage_scan.py`

Default DB path (see `object_detection/damage_scan/cli.py`):

`semi-labeling/step1_gdino_labeling/damage_scan/damage_scan.sqlite3`
